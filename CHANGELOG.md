# Changelog

## v0.1.0-spec
Initial public specification release of AMNION-ORACLE.

### Added
- Documentation-first architecture for a resonance wellness capsule prototype
- Safety state machine and stability metrics
- Software controller structure with runtime, safety gate, resonance model, LawX adapter, and ABRAXAS module
- Simulation-only I/O layer with sensor stub, actuator stub, and simulation runner
- Hardware documentation layer:
  - enclosure
  - acoustic cell
  - power system
  - integration
  - hardware-controller interface

### Scope
- Non-clinical
- Simulation-only
- No medical claims
- No treatment protocol
